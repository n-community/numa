About NUMA

Numa is an archive of player-created maps for the game N, an addictive platformer somewhat reminiscent of Lode Runner.

Numa provides a place for N map authors to post new maps they have created, so they can provide a simple URL. For players, it provides a centralised repository of maps to play, thumbnails of maps so players can get a glimpse of a map before loading it, and the ability to post comments and demo data for maps.

Currently, Numa provides no integrated method for posting a set of multiple maps, but this can be achieved by posting each map separately, then including the URLs to other maps in comments on each map.

Comments, complaints and suggestions are welcome, please email the webmaster with any and all of the above.

Multi-accounting

Having multiple accounts on NUMA is not allowed if they are both being used to place votes. Anyone breaking this rule will be dealt with harshly. Please contact us if you wish to have the name of your account changed.
